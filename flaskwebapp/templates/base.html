ML applications

Image Classification

A CNN model to classify hand-written digits. The example is adapted from tutorials on tensorflow official site. Network includes convolutional layer, Maxpooling layer, Dropout layer and Dense layer. The model is implemented with Keras.

Sentiment Analysis

A LSTM model to predict the rating based on comment texts. The model is built based on some yelp review datasets. The NLP workflow includes preprocessing (tokenization lemmatization and stop word removal), phrase modeling and lstm model implemented with Keras.
